Route

case class Route(from: Path, to: Option[Path], action: Option[(Conf, Seq[String]) ⇒ Seq[String]], children: List[PartialFunction[Path, Action]]) extends PartialFunction[Path, Action] with Product with Serializable

Route definition class.

from

Prefix for files, that will be matched.

to

Where should be matched files copied. If None, then file is ignored.

action

Transform to apply to the file. If None, then the file is copied as-is.

children

Child routes. If one of these routes matches the provided file, then the decision is delegated to it.
